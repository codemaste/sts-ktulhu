Seomon.com

Опубликовано 06.05.2014 автором admin

Website audit tool — https://seomon.com

Features:

  • website technologies, screenshot, common ranks (alexa, pagerank), social share counters (facebook, twitter, google+, pinterest, linkedin, reddit, vkontakte, odnoklassniki)
  • test website loading performance per request and per connection. Detect problems and provide recomendations
  • resolve domain DNS records via domain nameservers and root DNS servers, check availability and performance
  • check SSL certificate, validity, vulnerability, webserver SSL settings, supported SSL/TLS protocols
  • domain or ip whois information, registrar, name servers, contact details
  • test website for valid HTML code, display warnings and suggestions in syntax-highlited source
  • check domain and ips for listing in 300+ DNSBL and domain blacklists. Helps detect mail sending problems
  • find domains with same nameservers, mailservers, ip-addresses, ads and analytics ids
  • browse prevalence of frameworks, counters, widgets, ads, domain zones, mail, web and dns servers
  • inbound links of domain
